HILL CLIMB RACER - AUDIO FILES
===============================

All required audio files are present.

SOUND EFFECTS (.wav):
---------------------
[x] engine_idle.wav   : Engine idle loop
[x] engine_rev.wav    : Engine rev (speed-dependent)
[x] coin_collect.wav  : Coin pickup sound
[x] fuel_collect.wav  : Fuel can pickup
[x] crash.wav         : Crash/collision
[x] jump.wav          : Airborne sound
[x] land.wav          : Landing impact
[x] game_over.wav     : Game over jingle
[x] button_tap.wav    : UI button feedback
[x] upgrade.wav       : Upgrade success

MUSIC (.mp3):
-------------
[x] menu_music.mp3     : Main menu background
[x] gameplay_music.mp3 : In-game music

SETUP IN XCODE:
---------------
1. Open HillClimbRacer.xcodeproj in Xcode
2. Right-click on Resources folder > Add Files to "HillClimbRacer"
3. Select this Audio folder
4. Ensure "Copy items if needed" is checked
5. Ensure target membership includes HillClimbRacer
